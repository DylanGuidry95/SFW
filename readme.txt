Copy the sfwdraw.h and SFW.lib into your project directory.


In the solution explorer (visual studio),
	Right-Click Project
		-> Add
		-> Existing Item
		Select sfwdraw.h

	Right-click project
		-> Configuration Properties
		-> Linker
		-> Input
		-> Additional Dependencies
		-> Edit
		type in SFW.lib;

	
Look at the example project for usage.
